The site is a js frontend with apython backend. Everything is controlled by python
from now on.

1. Install python
2. create virtual environment by typing
    'python -m venv venv'
3. activate virtual environment by typing
    'venv\Scripts\activate'
4. Now the packages wont be installed to your whole system. but locally in your venv.
   So lets install them
    type 'pip install -r requirements.txt'
5. Ok, now rename the `example.env` to `.env` and replace with user,key and secret_key
   Tolis knows them.
